Ausführliche Beschreibung

Dieses Modul repräsentiert ein Code-Tabellen-Element also einen Eintrag in der Code-Tabelle. Ein solcher Eintrag enthält den Code mit seiner Code-Länge und das Zeichen, welches durch den Code repräsentiert wird.
