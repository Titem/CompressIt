CODETAB* read_codetab ( FILE *  input_stream)

Diese Funktion liest eine gespeicherte Code-Tabelle aus eine übergebenen Eingangs-Datenstrom (z.b. aus einer Datei), legt diese im Heap an und liefert einen Zeiger auf die neu angelegte Code-Tabelle zurück. Der übergebene Eingangs-Datenstrom muss an der Stelle stehen, an der die in der Datei gespeicherte Code-Tabelle beginnt.

Parameter
input_streamZeiger auf Datenstrom-Struktur, die die gespeicherte Code-Tabelle liefert
Rückgabe
Zeiger auf ausgelesene Code-Tabelle
