Ausführliche Beschreibung

Dieses Modul repräsentiert ein Huffman-Baum-Element also ein Bestandteil eines Huffman-Baumes. Ein solches Element kann entweder ein Knoten (htree_node) sein oder ein Blatt (htree_leaf). Ein Huffman-Baum-Element stellt in etwa eine abstrakte Generalisierung eines Huffman-Baum-Knotens und einem Huffman-Baum-Blattes da.

typedef struct S_HTREE_ELEMENT HTREE_ELEMENT

Dieser Datentyp repräsentiert ein Huffman-Baum-Element. Variablen diesen Datentyps beinhalten Huffman-Baum-Elemente.

Dokumentation der Aufzählungstypen

enum HTREE_ELEMENT_TYPE

Dieser Datentyp repräsentiert eine Auswahl um welchen Typ von Huffman-Baum-Element es sich handelt. Variablen diesen Datentyps können zwei Zustände annehmen, entweder Knoten oder Blatt.

Aufzählungswerte
NODE 

Es handelt sich um einen Knoten.

LEAF 

Es handelt sich um ein Blatt.
